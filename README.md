# drag-drop-test

A minimal Tauri example that accepts file drops on Windows 11 (built with Cursor).

## Prerequisites

- Windows 11
- Rust stable
- Trunk: `cargo install trunk`
- WASM target: `rustup target add wasm32-unknown-unknown`

## Run

```powershell
cargo tauri dev
```
## Notes
- Only Windows is tested.
